## Web Interface

This branch implements the ESP32 wireless communication module that enables data exchange between the web interface and the MCU. Below is a set of instructions on how to set up the web interface on a local device.

### Setup

1. Clone this branch to your local repository
2. Open the project in PlatformIO
3. Build and upload the main.cpp file to ESP32.
4. Open the Serial Monitor
5. Connect to the "IPR" Wi-Fi network (password: password)
6. Navigate to http://192.168.4.1/ in your internet browser to enable direct communication with the MCU.
